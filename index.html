Adversarial AI: Poking Holes in Classification Algorithms using GANs

Python - Generative Adversarial Networks-Text

Classification algorithms are an important aspect of our lives, from filtering our emails for spam to facial recognition models. But, with Adversarial Attacks, these models can be manipulated to produce incorrect classifications. An adversarial attack is an input specifically designed to cause misclassifications. This project analyzed the types of inputs that consistently cause misclassifications. Specifically, I created 3 machine learning models(RandomForest, XGB, DescisionTree) to classify financial fraud in a tabular dataset. Then, using Generative Adversarial Networks, I generated tabular data with the goal of fooling these models into classifying them as real. This dataset was used from Kaggle:Financial Dataset. This work was done during my internship with Oak Ridge National Laboratory, under the mentorship of Suhas Sreehari.
